Why feature scaling

  • feature scaling in machine learning is one of the most critical steps during the pre-processing
  • scaling can make a difference between a weak machine learning model and a better one
  • we dont want the higher number (scaling) to dominate in a model
  • the most common techniques of feature scaling are Normalization and Standardization

When to do feature scaling

  • essential for machine learning algorithms that calculate distances between data
  • if not scale, the feature with a higher value range starts dominating when calculating distances
  • in many algorithms, when we desire faster convergence, scaling is a MUST like in Neural Network
  • rule of thumb we may follow here is an algorithm that computes distance or assumes normality, scale your features

Algorithms where feature scaling matters

  • K-nearest neighbors
  • K-means
  • SVM
  • Principal Component Analysis(PCA)
  • gradient descent

Algorithms where feature scaling doesnt matter

  • these algorithms rely on rules
  • all tree-based algorithms
  • Linear Discriminant Analysis(LDA)
  • Naive Bayes

Normalization and standardization

  • normalization is used when we want to bound our values between two numbers, typically, between [0,1] or [-1,1]
  • standardization transforms the data to have zero mean and a variance of 1

To normalize or to standardize data

  • normalize when data is not normaly distributed

  • standardization when data is normaly distributed

  • standardization doesnt affect the outliers

  • normalization or standardization will depend on your problem and the machine learning algorithm you are using

How to perform feature scaling

  • Min Max Scaler
  • Standard Scaler
  • Max Abs Scaler
  • Robust Scaler

Min Max scaler

  • m = (x -xmin) / (xmax -xmin)

Solution

#Python
def min_max_normalization(values):
    minx=min(values)
    maxx=max(values)
    return [(x-minx)/(maxx-minx) for x in values]
#R
normalizeF<-function(x) {return((x-min(x))/(max(x)-min(x)))}

  • m=(X )/

Lab

  • create UDF for standardization in Python and R

Solution

#Python
import statistics as s
def standardization(values):
    meanx=s.mean(values)
    stdx=s.stdev(values)
    return [(x-meanx)/stdx for x in values]
#R
standardizeF<-function(x) {return((x-mean(x))/sd(x))}
